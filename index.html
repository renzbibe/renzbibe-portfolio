To-do List

Create a to-do list for multiple projects, which are stored locally in the user's browsers. Furthermore, you are able to modify the name of the projects and the items in the list if you wish to do so!

Process: I created a two-dimensional array to create multiple projects and store its contents - the tasks in the todo-list. The projects are accessible using the first subscript of the two-dimensional array, while the tasks in the projects are accessible using the second subscript of the two-dimensional array. I utilised the splice method to remove projects and tasks in the array. Additionally, I made use of event targets to specify which projects or tasks are to be edited or removed. I also made use of the local storage object in JavaScript to save the projects locally in the user's browser. Furthermore, I selected a darker colour scheme to reduce eye strain to the users.

Skills: HTML, CSS, JavaScript

Rock, Paper, Scissors Game

A rock, paper, and scissors game where you play against a computer, implemented with a score system - keep track of your wins, ties and of course losses!

Process: The algorithm of the computer you play against is implemented using the Math object in JavaScript to generate a random number ranging from 1 to 3. This will decide the computer's move whether it is Rock, Paper or Scissors accordingly. I used if conditions to make this work and also to display the appropriate images for the player and computer's move. Furthermore, I implemented a score system to keep track of the wins, losses and ties of the user stored locally in the user's browser using the local storage object in JavaScript.

Skills: HTML, CSS, JavaScript
